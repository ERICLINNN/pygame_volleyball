# Pixel Monster Smash 

Pixel Monster Smash is a local two player competitive volleyball game.

## Installation

1. Make sure you have **Python 3.12** installed（Pygame may not always support the latest Python version immediately）.

2. Install the Pygame library:

### Windows
```bash
python -m pip install pygame
```

### macOS
```bash
python3 -m pip install pygame
```


## Controls
Grab a friend and compete on the same keyboard!

### Player 1 (Purple Monster)
- **W**: Jump
- **A / D**: Move Left / Right
- **G**: **SMASH!** (Must be in mid-air when contacting the ball)

### Player 2 (Grey Monster)
- **Up Arrow**: Jump
- **Left / Right Arrow**: Move Left / Right
- **Enter / Return**: **SMASH!** (Must be in mid-air when contacting the ball)
